# Data Analyst Agent

A conversational AI agent that lets you upload a CSV or Excel file and ask questions about your data in plain English. Every answer is backed by real Python/pandas execution — no guessing.

## Demo

Upload a dataset → ask a question → get a table or chart generated by actual code.

```
User: What is the average sleep hours by gender?
Agent: [runs pandas code] Males average 6.8h, females average 7.1h.

User: Plot that as a bar chart.
Agent: [runs matplotlib code] [chart appears inline]
```

## Features

- **Natural language Q&A** — powered by Claude's tool-use API
- **Real code execution** — agent writes and runs Python/pandas against your data
- **Inline charts** — matplotlib/seaborn figures rendered directly in chat
- **Auto-retry on errors** — if generated code crashes, agent reads the traceback and fixes it
- **CSV and Excel support** — `.csv`, `.xlsx`, `.xls`
- **Safe execution** — DataFrame is never mutated; each run gets a fresh copy

## Architecture

```
User question
  └─▶ build_system_prompt(schema)          # inject column names, types, sample rows
  └─▶ Claude API (tool-use)
        ├─ stop_reason == end_turn  ──▶ return final text
        └─ stop_reason == tool_use
              └─▶ execute_python(code, df)  # sandboxed exec()
              └─▶ append tool result to history
              └─▶ loop (max 5 iterations)
```

**Key design choices:**
- Bounded ReAct loop (max 5 iterations) prevents runaway API costs
- `df.copy()` inside the executor prevents agent code from mutating the user's data
- Full message history replayed on every API call (Anthropic's stateless API requirement)
- `FakeLLMClient` test double — all 33 tests run without real API calls

## Getting Started

### Prerequisites

- Python 3.11+
- An [Anthropic API key](https://console.anthropic.com/)
- Windows users: enable [Win32 Long Paths](https://pip.pypa.io/warnings/enable-long-paths) before installing

### Install

```bash
git clone https://github.com/MonkeyJeremy/data-analyst-agent.git
cd data-analyst-agent
pip install -e ".[dev]"
```

### Configure

```bash
cp .env.example .env
# edit .env and set ANTHROPIC_API_KEY=sk-ant-...
```

### Run

```bash
python -m streamlit run app.py
```

Or enter your API key directly in the sidebar — no `.env` needed.

### Test

```bash
pytest --cov=src --cov-report=term-missing
```

## Security Note

This project uses `exec()` to run agent-generated Python code locally. It is intended for **local development only**. For production deployment, replace the executor with a sandboxed environment such as [E2B](https://e2b.dev/).
